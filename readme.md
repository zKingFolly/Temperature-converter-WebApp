A simple temperature converter project.

This is a vanilla JavaScript project intended for learning and practice with basic DOM manipulation and unit conversion logic.

This project provides a small web app that converts temperatures between Celsius, Fahrenheit, and Kelvin.

## Features

- Convert values between Celsius, Fahrenheit, and Kelvin
- User-friendly interface with instant results
- Built with HTML, CSS, and JavaScript

## Usage

1. Enter a temperature value.
2. Select the input unit.
3. View the converted temperatures.
